Angular drills

Part 1: working in the view

   The aim of "Angular drills part 1" is to focus on the view by using native angularJS directives, to play with UI states and to resist the temptation to use controllers or directives.
   During part one, all drills have to be performed without the use of any controller or custom directive.

- Display the number of countries visible
- Only display a continent name if there is a list of countries to display for it (in other words: if the search doesn't return any result for a continent, don't display its name!)
  • Ng-show directive: http://docs.angularjs.org/api/ng.directive:ngClick
  • Filters: http://docs.angularjs.org/api/ng.filter:filter
  • Ng-pluralize directive: http://docs.angularjs.org/api/ng/directive/ngPluralize
